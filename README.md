# Updated README for Interview Bot Project

This project aims to provide a comprehensive platform for interview preparation with features to simulate interviews, provide feedback, and track progress.

## Features
- Mock interviews based on various topics.
- Feedback on performance.
- Progress tracking.

## Installation
To install this project, clone the repository and run the setup script.

```bash
git clone https://github.com/mohit-756/interview_bot_project_1.git
cd interview_bot_project_1
```

## Usage
Run the main application with the command:

```bash
python app.py
```
